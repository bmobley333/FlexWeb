Flex Moxie - First-Time Player Setup Guide
==========================================


Welcome to the Flex Moxie Campaign Engine! Follow these steps to initialize your personal player workspace and Codex sheet.


1. Copy the Codex Template
--------------------------
Make a copy of the 'FlexMoxie - Codex' spreadsheet from the shared campaign folder to your own personal Google Drive.


2. Initialize and Authorize the Setup Script
--------------------------------------------
- Open your copied Codex spreadsheet.
- Wait 5-10 seconds for the spreadsheet custom menus to load.
- Locate the custom menu named '💪 FlexMoxie' in the top menu bar.
- Click '💪 FlexMoxie' ➔ '▶️ Activate 💪FlexMoxie Menus'.
- A standard Google Apps Script authorization prompt will appear requesting permission to access your Drive files and sheets. Grant the permissions to allow the automation to run.


3. Automated Directory Organization
-----------------------------------
Once authorized, the script will run automatically to establish your game folder environment:
- It will create a parent directory named '🫀FlexMoxie' in your root Google Drive.
- It will create three subfolders: 'Master Copies', 'Characters', and 'Custom Abilities'.
- It will move your Codex spreadsheet into the parent '🫀FlexMoxie' directory and rename it to 'Player's Codex'.
- It will recursively copy all player-required reference manuals, character logs, and sheet templates from the master library into your 'Master Copies' subfolder.


4. Reload & Start Playing
-------------------------
When the popup alert states 'Setup Complete! Please refresh browser tab', click OK and refresh your browser tab. Your Codex will reload with the full custom gameplay menus ready to use.
